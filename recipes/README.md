# Recipe Database 🍳🐟🥗

A growing collection of simple, filling, Basque/Mediterranean-friendly recipes — the kind you can
actually make and enjoy while losing weight. Every recipe is built on the guide's principles:
**protein + vegetables first, measured oil, real food, big volume for the calories.**

## How recipes are tagged

Each recipe card carries quick tags so you can pick by mood, time, or path:

- **Meal:** breakfast · lunch · dinner · snack
- **Path:** 🫒 Mediterranean · 🍳 Low-carb/High-protein (many fit both)
- **Mood/craving:** "feel like yogurt", "feel like eggs", "feel like fish", "quick", "comfort",
  "sweet-tooth-friendly"
- **Rough calories & protein** (per serving, approximate — for awareness, not obsession)

> In the interactive site these become **filterable cards** ("Today I feel like… yogurt" → shows
> matching recipes). On paper, they're a simple recipe booklet.

## Notes

- Calorie/protein numbers are **estimates** to build awareness — real values depend on portions.
- "Measure the oil" applies everywhere: 1 tablespoon ≈ 120 kcal. Use a spoon or a spray.
- Most recipes scale easily — make extra protein and vegetables for tomorrow's lunch.
- More recipes get added over time; this is a living database.
