## Design note (v1)

The interactive site is built to look polished **without** requiring downloaded photos — it uses a
clean color system, food emoji, and CSS illustration so it stays a single, offline, shareable file.
Photo "slots" are marked in the build so real images can be dropped in later without redesigning.
